# 🌸 AI-Based Iris Flower Species Classification

A machine learning project that classifies Iris flowers into **Setosa**, **Versicolor**, and **Virginica** using the **K-Nearest Neighbors (KNN)** algorithm. The project demonstrates the complete machine learning workflow, including data exploration, visualization, preprocessing, model training, evaluation, and prediction.

---

## 📌 Project Overview

This project uses the famous **Iris Dataset** from Scikit-learn to build a multiclass classification model.

The workflow includes:

- Data Loading
- Exploratory Data Analysis (EDA)
- Data Visualization
- Feature Scaling
- Model Training
- Model Evaluation
- Prediction on New Data

---

## 🚀 Features

- Exploratory Data Analysis (EDA)
- Count Plot of Flower Species
- Pair Plot for Feature Relationships
- Correlation Heatmap
- Train-Test Split (80:20)
- Feature Scaling using StandardScaler
- K-Nearest Neighbors (KNN) Classification
- Accuracy Score
- Confusion Matrix
- Classification Report
- Weighted F1 Score
- Prediction on New Flower Samples

---

## 📊 Machine Learning Workflow

1. Load the Iris Dataset
2. Create a Pandas DataFrame
3. Perform Data Exploration
4. Visualize the Dataset
5. Split Data into Training and Testing Sets
6. Scale Features using StandardScaler
7. Train the KNN Classifier
8. Evaluate Model Performance
9. Predict Species for New Samples

## ▶️ Installation

Clone the repository:

```bash
git clone https://github.com/returnzerovicky/iris-flower-classification.git
```

Move into the project folder:

```bash
cd iris-flower-classification
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the project:

```bash
python iris_classifier.py
```
